# v0.1.4

- Removed the original SDL2 render backend — D3D11 pipeline only.
- D3D11 zero-copy path (NV12).
- HDR P010 support.
- Software decode fallback:
    - `video_thread.c`: HW stall watchdog → SW fallback.
    - `video_renderer_d3d11.h`: color-param cbuffer + target enum.
    - NV12 shader: PQ/HLG inverse EOTF, BT.2020→BT.709, BT.2390 EETF, sRGB encode.
    - cbuffer populated from `AVFrame` metadata + side data (mastering nits).

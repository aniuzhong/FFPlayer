# v0.1.0

Initial `ffplay.exe` build without compiling FFmpeg from source.

> Vulkan/libplacebo is excluded — wiring it on Windows is impractical.

## Layout

```
third_party/
    ffmpeg-8.1-full_build-shared/   # Gyan bundle (include, lib, bin, ffplay.exe)
    SDL2-2.32.10/                   # SDL2 MSVC dev package
    .cache/                         # CMake download cache
src/
    ffplay.c, cmdutils.c, ...       # Source snapshot aligned with upstream fftools
    compat/                         # va_copy.h, w32dlfcn.h
    shim/include/libavutil/         # Stub headers + include adaptations
cmake/
    FFmpegGyan.cmake                # config generation + configuration probing
    config.h.in                     # libplacebo disabled (no Vulkan render path)
    config_components.h.in
```

## Build

```powershell
cmake -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release

.\build\Release\ffplay.exe -version
```
